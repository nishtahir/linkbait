# Hey Snackfood Plugin

Inspired by [Hey taco](https://www.heytaco.chat/), Hey snackfood is a fun way to reward
your teamates delicious treats for being awesome.

# Usage

Simply @ mentioning someone and add :nutella:. You can give multiple :nutella: at once.

Request the leaderboard using the `leaderboard` command.
